Kickboard, le bookmarklet

Version : 1.2

À quoi ça sert ?

Kickboard est une collection de boutons permettant de chatter rapidement pendant les concerts sur Twitch du groupe Kickban.

Les boutons sont regroupés par chanson et contiennent des paroles ou des gimmicks revenant dans les chansons de Kickban.

Installation

Pour installer Kickboard, le bookmarklet, faites un glisser-déposer du lien ci-dessous dans votre barre de favoris.

Kickboard

Ne cherchez pas à visiter le lien depuis cette page, il ne pourra fonctionner qu’une fois installé dans votre barre de favoris.

Utilisation

Pour l’utiliser, rendez-vous sur la page Twitch de Kickban, affichez le chat et cliquer sur le lien Kickboard dans votre barre de favoris.

Il ne vous reste plus qu’à sélectionner une chanson et vous pourrez chatter en cliquant simplement sur un bouton.

La Kickboard reste installée sur la page tant que vous ne fermez pas votre fenêtre ou que vous ne rafraîchissez pas la page.

Désinstallation

Pour désinstaller un bookmarklet, c’est très simple : il suffit de le supprimer de votre barre de favoris !

Mise à jour

Pour mettre à jour un bookmarklet, il faut d’abord le désinstaller pour ensuite installer la nouvelle version.

Pour les bidouilleureuses

Tout le code est disponible sur le dépôt Git du Kickboard.
